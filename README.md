# Classificação de Vinhos com Machine Learning

[Python 3.9+]

Este projeto propõe utilizar dados físico-químicos e técnicas de ML para auxiliar na previsão, e desenvolve um modelo de Machine Learning capaz de prever a qualidade de vinhos tintos (Alta ou Baixa/Média) com base em suas características físico-químicas.

**Objetivo Principal:** Desenvolver um modelo de classificação binária 
(Alta Qualidade: nota ≥ 7 ou Baixa/Média: nota < 7).

Wine Quality Dataset (Kaggle)
(https://www.kaggle.com/datasets/yasserh/wine-quality-dataset)

**Visão Geral**
- Total de amostras: 1018 (após limpeza)
- Features: 11 variáveis físico-químicas
- Classes: 2 (Alta Qualidade e Baixa/Média Qualidade

Ferramentas utilizadas:
- Python 3.9+
- pandas, numpy, scikit-learn
- matplotlib, seaborn
- Jupyter Notebook

Principais resultados:
Melhor Modelo: Random Forest
- Acurácia: 91.67%
- Precisão: 77.78%
- F1-Score: 62.22%
- ROC-AUC: 90.64%
